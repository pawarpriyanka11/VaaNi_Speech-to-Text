About VaaNi

VaaNi is a browser-native speech-to-text demo with real-time microphone recognition and file-based ASR support. It works with live audio and uploaded files (wav/mp3/any audio supported by the cloud endpoint or browser codec).

The app can use Web Speech API locally and cloud inference APIs for uploaded files, so it is not limited to a fixed pretrained dataset corpus.

Capabilities

  • 3 languages: Hindi, Marathi & English
  • Web Speech API + Whisper inference for multilingual speech recognition
  • Automatic stereo-to-mono conversion and 16kHz resampling
  • Word Error Rate (WER) and accuracy scoring
  • Real-time microphone recording support
  • Compatible with .wav and .mp3 formats
